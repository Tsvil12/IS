# Погодный бот
---
### p.s. Делал этот бот до блокировок, все работало
---
Этот бот умеел показывать погоду в любом городе.  
Он получает данные из разных источников: через API OpenWeatherMap и с сайтов Яндекс.Погоды и Gismeteo.

---

## Как пользоваться

1. в Telegram: **@TsvilBOt**
2. Отправь команду `/start` — появится список доступных команд
3. Выбери нужную команду и укажи город

---

## Команды

| Команда | Пример | Что делает |
|---------|--------|-----------|
| `/start` | `/start` | Показывает приветствие и список команд |
| `/weather ` | `/weather Кемерово` | Погода сейчас (данные с OpenWeatherMap) |
| `/forecast ` | `/forecast Кемерово` | Прогноз на 3 дня (OpenWeatherMap) |
| `/yandex ` | `/yandex Кемерово` | Погода с Яндекс.Погоды (парсинг) |
| `/gismeteo ` | `/gismeteo Кемерово` | Погода с Gismeteo (парсинг) |

---

## Пример

```
/weather Москва

🌡 Погода в Кемерово:
Температура: 15°C
Описание: небольшая облачность
Влажность: 78%
```
